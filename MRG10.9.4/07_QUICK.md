```
[Genre] [Subgenre], [Artist DNA Reference], [Tempo]bpm, [Key], [Vocal Style], [Production Style], [Mood/Atmosphere], [Instrumentation], [Era Reference], [Special Elements]
```

#### Music Quick Formula (Safe Mode — ใช้เมื่อ User รีบ)

> ตัดความซับซ้อนออก เหลือแค่เนื้อๆ (ข้ามขั้นตอน 10 Concepts):

```text
[Genre], [Vocal Gender], [Tempo]bpm, [Mood], [Key Instrument]
```

ตัวอย่าง: `Pop Rock, Energetic, 120bpm, C Major, Male Vocals, Electric Guitar, 2000s`

#### Genre Quick Templates (แม่แบบสำเร็จรูป)

| Genre | Template |
|---|---|
| **Pop** | Pop, Dance Pop, upbeat, 120bpm, C Major, bright vocals, polished production, feel-good, synth-driven |
| **Rock** | Rock, Alternative Rock, driving, 130bpm, E Minor, powerful vocals, guitar-heavy, energetic, distorted guitars |
| **Thai Pop** | Thai Pop, Indie Pop, melodic, 110bpm, G Major, emotional Thai vocals, modern production, bittersweet, acoustic guitars |
| **Luk Thung (ลูกทุ่ง)** | Luk Thung, Modern Mor Lam, traditional, 95bpm, Thai scale, nasal vocals, khaen, nostalgic, Isaan flavor |

#### Key + Tempo Guide (ตารางคู่สีอารมณ์)

| อารมณ์ | Key | Tempo |
|---|---|---|
| **Happy/Upbeat** | C, G, D, F Major | 115-130 bpm |
| **Sad/Emotional** | A, E, B Minor | 60-90 bpm |
| **Dark/Intense** | F#, C#, G# Minor | 130-150 bpm |

#### Essential Metatags (ก๊อปไปวางได้เลย)

```
[Intro] [Verse] [Pre-Chorus] [Chorus] [Bridge] [Solo] [Outro] [End]
```

### 2. VIDEO DOMAIN: THE CINEMATIC CHEAT SHEETS

คัดเฉพาะ "ศัพท์เทคนิค" ที่ AI วิดีโอเข้าใจ 100%:

#### Shot Type Cheat Sheet (ระยะภาพ)

| Shot | ชื่อเต็ม | ใช้เมื่อ |
|---|---|---|
| **ECU** | Extreme Close-Up | เน้นอารมณ์/รายละเอียด |
| **CU** | Close-Up | เน้นสีหน้า/Expression |
| **MS** | Medium Shot | เน้น Action/การกระทำ |
| **WS** | Wide Shot | เน้น Setting/สถานที่ |
| **EWS** | Extreme Wide Shot | เน้นความอลังการ/Epic |

#### Camera Movement Cheat Sheet (การเคลื่อนที่)

| Movement | ผลลัพธ์ |
|---|---|
| **Dolly In** | สร้างความใกล้ชิด/โฟกัสอารมณ์ |
| **Whip Pan** | สร้างพลังงานและการเปลี่ยนฉาก |
| **Dutch Angle** | สร้างความรู้สึกไม่ปลอดภัย/วุ่นวาย |
| **Pan** | เผยให้เห็นสิ่งใหม่ (Reveal) |
| **Track** | ติดตามตัวละคร (Follow) |

#### Video Quick Formula (Safe Mode — กันตายสำหรับวิดีโอ)

```text
Cinematic shot of [Subject], [Action], [Lighting], 4k --ar 16:9
```

ตัวอย่าง: `Medium shot of a cyberpunk girl hacking, neon lighting, anime style, 8k --ar 16:9`

### 1. MUSIC ERRORS

| Error Code | ปัญหา | วิธีแก้ |
|---|---|---|
| **M001** | Artist Name in Prompt — AI ปฏิเสธ | เปลี่ยนชื่อศิลปินเป็น DNA Description (Vocal + Production + Mood) |
| **M002** | Rhythm Guessing — ไม่ระบุ BPM/Key หรือเดาเอาเอง | บังคับ Research 3 แหล่ง (Hooktheory, Tunebat, SongBPM) |
| **M003** | Missing Structure Tags — เพลงมั่ว | ต้องเติม [Intro], [Verse], [Chorus] ล็อคไว้ทุกครั้ง |
| **M004** | Language Mismatch — ต้นฉบับภาษาหนึ่ง แต่ AI ร้องอีกภาษา | ล็อกภาษาตามต้นฉบับ 100% หรือใช้ Phonetic Lock |
| **M005** | DNA Hallucination — เนื้อเสียงไม่ตรงกับ Artist DNA | กลับไป Re-check [TIER-XX] Reference ในไฟล์ 02 |
| **M006** | Thai Lyrics Tone Mismatch — ร้องเพี้ยน | เช็คการวางวรรณยุกต์ให้ตรงกับทิศทาง Melody (Ascending/Descending) |

### DECODED: Music Troubleshooting (คู่มือแก้บั๊กเพิ่มเติม)

| อาการ | สาเหตุ | วิธีแก้ (Solution) |
|---|---|---|
| ร้องไม่ชัด (Mumbling) | ใส่เนื้อเพลงเยอะเกินไปใน 1 บรรทัด | แบ่งท่อนเพลงใหม่ ให้สั้นลง |
| เสียงเพี้ยน (Artifacts) | ใส่ Style Strength สูงเกินไป | ลดความซับซ้อนของ Prompt |
| ภาษาไทยแปร่งๆ | ไม่ได้ใส่วรรณยุกต์กำกับ | ใช้ Phonetic (คำอ่านอังกฤษ) ช่วย + ใส่ tag [Clear Diction] + เว้นวรรคคำ |
| เพลงไม่จบ | ลืมใส่ Tag จบ | เติม [Outro] และ [End] ท้ายสุด / หรือ [Fade Out] |
| แนวเพลงไม่ตรง | ใส่ Genre ขัดแย้งกัน | เลือก Genre หลักแค่อันเดียวพอ |
| เสียงดนตรีตีกันมั่ว (Muddy Mix) | เครื่องดนตรีเยอะไป | ลดจำนวนเครื่องดนตรีใน Prompt ออก (Less is More) |

### 3. SEO ERROR CODES (S001-S006)

| Code | Error Scenario | Solution (วิธีแก้) |
|---|---|---|
| **S001** | Title Over 60 Chars | ตัดคำสร้อยออก ให้ Keyword หลักอยู่ 1-3 คำแรก |
| **S002** | Keyword Stuffing | ใช้คำที่ใกล้เคียง (LSI Keywords) แทนการพิมพ์คำเดิมซ้ำๆ |
| **S003** | No Hook in First 200 Chars | บรรทัดแรกของ Desc ต้องมีคำที่หยุดคนอ่าน (เช่น "แจกฟรี", "วิธีลับ") |
| **S004** | Missing Timestamps | บังคับใส่ Timestamps ทุกคลิปที่ยาวเกิน 3 นาที |
| **S005** | Generic Tags | ลบแท็กกว้างๆ (เช่น #AI, #Music) แล้วใส่แท็กเฉพาะเจาะจงแทน |
| **S006** | No CTA in Description | เพิ่มประโยคปิดท้าย "กด Subscribe เพื่อ..." หรือ "สอบถามได้ที่..." |

### 4. CRITICAL RECOVERY PROTOCOL (ขั้นตอนกู้ชีพงาน)

> นี่คือส่วนที่สำคัญที่สุดของไฟล์! เมื่อ AI สติแตก (Hallucinate) หรือ User ด่าว่า "ทำไมทำไม่ได้?" ไฟล์นี้มี Script ตอบกลับที่ผ่านการจิตวิทยามาแล้ว:

เมื่อผลลัพธ์ล้มเหลวโดยสิ้นเชิง ระบบจะทำดังนี้:

| Step | Action | รายละเอียด | ตัวอย่าง Script |
|---|---|---|---|
| 1 | **Acknowledge (ยอมรับ)** | ห้ามเถียง! ยอมรับว่าพลาด | "ขออภัยครับ ผลลัพธ์เมื่อกี้มีความผิดพลาดเรื่อง [ระบุปัญหา]" |
| 2 | **Diagnose (วินิจฉัย)** | วิเคราะห์ว่าพลาดที่ Prompt, Format หรือการประมวลผล | "สาเหตุเกิดจาก Prompt ซับซ้อนเกินไป / คีย์ไม่ตรงกับเสียงร้อง" |
| 3 | **Correct (แก้ทันที)** | ห้ามถามว่า "ให้แก้ไหม?" แต่ให้ส่งตัวแก้ไปเลย | "ผมได้ปรับ Prompt ใหม่โดยการ [ลดคำสั่ง / เปลี่ยนคีย์] ครับ" |
| 4 | **Re-Deliver (ส่งใหม่)** | ส่ง Output ตัวใหม่ทันที ห้ามรอถาม | (ส่ง Output ตัวใหม่ทันที) |

## PART C: FINAL QC CHECKLIST (เช็คก่อนปล่อยของ)

### 5-Point Pre-Flight Check (ก่อนกดส่งให้ User)

- [ ] **1. COPY-PASTE READY:** Output อยู่ใน Code Block พร้อมก๊อปวางไหม?
- [ ] **2. NO HALLUCINATIONS:** ไม่ได้มั่ว BPM หรือชื่อศิลปินใช่ไหม?
- [ ] **3. THAI CHECK:** (ถ้าเป็นภาษาไทย) เช็คสระลอย/วรรณยุกต์หรือยัง?
- [ ] **4. FORMAT:** เรียง Lyrics → Style → Title ถูกต้องไหม?
- [ ] **5. TONE:** น้ำเสียงการตอบเป็น "MrArranger" (มืออาชีพ) ไหม?

### Platform-Specific Checks

- [ ] **Platform Check:** เลือกเครื่องมือถูกไหม? (Suno vs Udio vs Runway vs MJ)
- [ ] **Syntax Check:** มี Code Block ครอบไหม?
- [ ] **Language Check:** ภาษาตรงตามต้นฉบับไหม?
- [ ] **Artist Check:** ไม่มีชื่อศิลปินหลุดไปใช่ไหม?
- [ ] **Logic Check:** BPM/Key เป็นไปได้จริงไหม?

### Legacy QC

- [ ] No placeholder text [like this]
- [ ] Code blocks formatted correctly
- [ ] Language matches artist origin
- [ ] Description Line 1 matches Title 100%

### ERROR RECOVERY (THE FIXES)
- **M001 (Artist Name):** Replace with DNA keywords (vocal style + instruments)
- **M002 (Rhythm Guessing):** Force Research 3 sources (Hooktheory, Tunebat, SongBPM)
- **M004 (Language Mismatch):** Lock language to original 100% or use Phonetic Lock
- **M005 (DNA Hallucination):** Re-check [TIER-XX] Reference in file 02
- **M006 (Thai Tones):** Match melody contour (Up = Rising tone / Down = Falling tone)
- **V002 (Lighting Inconsistency):** Use Lighting Table in file 03 (match Mood)
- **V003 (Consistency):** Create Character Lock Prompt and apply to ALL shot prompts
- **S001 (Long Title):** Trim to 50-60 chars, Keyword in first 1-3 words
- **S002 (Keyword Stuffing):** Use LSI Keywords instead of repeating
- **S003 (No Hook):** Place hook in first 150 chars of Description
- **S004 (No Timestamps):** Mandatory for videos > 3 mins
- **S005 (Generic Tags):** Use specific niche tags > broad tags
- **S006 (No CTA):** Ensure "Click/Sub/Follow" is present
- **Crash Protocol:** Acknowledge → Diagnose → Correct → Re-Deliver (ห้ามถามกลับ ส่งตัวแก้เลย)
