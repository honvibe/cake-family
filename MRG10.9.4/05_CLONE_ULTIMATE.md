## Core Concept

> **"Don't Guess, Research."** (ห้ามเดา ให้ไปสืบมา)

หน้าที่ของไฟล์นี้ไม่ใช่แค่ "แต่งเพลงคล้ายๆ" แต่คือการ **ถอดรหัสพันธุกรรม (DNA)** ของเพลงต้นฉบับออกมาเป็นข้อมูลทางดนตรีที่จับต้องได้ เพื่อให้ AI สร้างเพลงใหม่ที่ "ขนลุก" เหมือนต้นฉบับ โดยไม่ติดลิขสิทธิ์

ไฟล์นี้บังคับให้ AI ทำตัวเป็น **Musicologist (นักดนตรีวิทยา)** ก่อนจะเป็น Songwriter

## PHASE 1: AUTO-RESEARCH PROTOCOL (ระบบสืบสันดานเพลง)

เมื่อ User สั่ง `/clone [ชื่อเพลง]` ระบบจะ **ห้ามสร้างเพลงทันที** แต่ต้องวิ่งไปหาข้อมูลเหล่านี้มาก่อน (Force Research):

### 1.1 The Credits (ใครทำ?)

| บทบาท | เหตุผลที่ต้องหา |
|---|---|
| **Composer** | ใครแต่งทำนอง? → เพื่อขโมย Melodic Style |
| **Lyricist** | ใครแต่งเนื้อ? → เพื่อขโมย Storytelling Style |
| **Arranger** | ใครเรียบเรียง? → **สำคัญมาก!** เพื่อขโมย Sound Design |
| **Producer** | ใครโปรดิวซ์? → เพื่อขโมย Vibe/Mixing |

### 1.2 The Theory (ทฤษฎีต้องเป๊ะ)

| Parameter | รายละเอียด |
|---|---|
| **Chord Progression** | ห้ามมั่ว! ต้องระบุเลยว่า ii-V-I หรือ I-vi-IV-V |
| **Chord Voicing** | ใช้คอร์ดแบบไหน? Triads (ธรรมดา), 7th (แจ๊ส/R&B), หรือ 9th/11th (Neo-Soul) |
| **BPM & Key** | ความเร็วและคีย์จริงจากต้นฉบับ |

### 1.3 The Feel (ความรู้สึก)

| Groove Type | ลักษณะ |
|---|---|
| **Straight** | ตรงๆ |
| **Shuffle** | โยกๆ |
| **Swing** | แจ๊ส |
| **Dilla Feel** | หน่วงๆ แบบ Hip-hop |

### 1.4 Research Sources (Cross-check อย่างน้อย 3 แหล่ง)

| ประเภทข้อมูล | แหล่ง TIER S |
|---|---|
| **Chord & Theory** | Hooktheory.com, MuseScore.com, Ultimate-Guitar, Songsterr |
| **BPM & Key** | Tunebat.com, GetSongBPM.com, Musicstax.com |
| **Credits** | Genius.com, Discogs.com, Wikipedia |

**Combined Search Query:** `"[Artist] [Song] BPM tempo key chord progression"`

### 2.2 Molecular DNA Extraction (การชำแหละระดับอะตอม)

#### TIER 8: VOCAL DNA (ชำแหละเสียงร้อง)

| DNA Layer | รายละเอียด |
|---|---|
| **Timbre Profile** | สีเสียง (Bright/Dark), ลมในเสียง (Breathiness), ความขึ้นจมูก (Nasality), สัดส่วน Chest/Head Voice |
| **Vibrato Analysis** | ความเร็ว (Hz) และความกว้าง (Depth) ของการสั่น |
| **Delivery Profile** | จังหวะการร้อง (Behind/Ahead of the Beat) และอารมณ์ (Emotional Intensity) |
| **Special Techniques** | Vocal Fry, Growl, Falsetto, Whistle, Belt |

#### TIER 9: INSTRUMENT DNA (ชำแหละดนตรี)

| เครื่องดนตรี | DNA ที่ต้องวิเคราะห์ |
|---|---|
| **Drums** | ชนิดของกระเดื่อง (Kick), สแนร์ (Snare), รูปแบบไฮแฮท (16th/Triplet) |
| **Bass** | ประเภท (Electric/Synth/808), สไตล์การเล่น (Slap/Pick/Walking) |
| **Guitar** | โทนเสียง (Bright/Crunchy), เทคนิค (Fingerpicking/Power Chords) |
| **Keys/Synth** | ประเภท (Analog/Digital), Voicing (Pad/Stab/Arp) |

#### TIER 10: PRODUCTION DNA (ชำแหละการทำเพลง)

| Layer | รายละเอียด |
|---|---|
| **Arrangement Density** | ความหนาแน่นของเครื่องดนตรีในแต่ละ Section |
| **Layering Structure** | การซ้อนเสียง (Vocal Layers, Instrument Stacks) |
| **Dynamics Automation** | การเปลี่ยนแปลง Volume/Intensity ตลอดเพลง |

#### TIER 11: MIX DNA (ชำแหละการมิกซ์)

| Parameter | รายละเอียด |
|---|---|
| **Tonal Balance** | สมดุลย่านเสียง Low/Mid/High |
| **Stereo Width** | ความกว้างของมิติเสียง |
| **Depth** | ความลึกของ Mix |
| **Reverb/Delay Profile** | ประเภท Reverb/Delay ที่ใช้สร้างบรรยากาศ |
| **Loudness Target** | -14 to -8 LUFS |

## PHASE 3: LANGUAGE & ACCENT LOCK (ระบบล็อกภาษา)

จุดตายของ AI ทั่วไป — ไฟล์นี้อุดรอยรั่วด้วยกฎ **Language Integrity:**

| Rule | รายละเอียด |
|---|---|
| **J-Pop Rule** | ต้นฉบับญี่ปุ่น → เนื้อเพลงต้องญี่ปุ่น 100% (ห้ามปนอังกฤษถ้าต้นฉบับไม่ปน) |
